## Querying DAOhaus v2 Data for Mainnet
DAOhaus' v2 data for Mainnet are available through Subgraph Studio. To query DAOhaus v2 data, please visit our [Subgraph Studio Link here](https://thegraph.com/explorer/subgraph?id=9uvKq57ZiNCdT9uZ6xaFhp3yYczTM4Fgr7CJHM6tdX9H&view=Indexers)

If you are new to Subgraph Studio, you will need to:  
1. Create your API keys at [Subgraph Studio API Keys](https://thegraph.com/studio/apikeys/) (Comes with 1,000 free API calls)
2. Fund your billing balance (if needed)s
3. Query the DAOhaus v2 Subgraph via `https://gateway.thegraph.com/api/[api-key]/subgraphs/id/9uvKq57ZiNCdT9uZ6xaFhp3yYczTM4Fgr7CJHM6tdX9H`

Facing any issues? Hop into our [Discord](https://discord.gg/invite/daohaus)'s' and give `@arentweall` and `@UI369` a ping! We will help you get set up with whatever you need.
